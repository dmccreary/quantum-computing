# List of MicroSims

This textbook contains **51 interactive MicroSims** — browser-based simulations
that let you explore quantum computing concepts, investment dynamics, cognitive
biases, and critical thinking frameworks hands-on.
Each MicroSim runs entirely in your browser with no installation required.
