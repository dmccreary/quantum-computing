# Bias Compounding Cascade

This MicroSim demonstrates how cognitive biases do not operate in
isolation -- they compound. Each bias amplifies the distortion
introduced by the previous one, creating a cascade effect that can
push an investor's perceived probability of success far beyond what
the evidence supports.

The simulation walks through five common biases that affect quantum
computing investment decisions: anchoring bias, authority bias, the
bandwagon effect, confirmation bias, and the sunk cost fallacy. Each
stage shows its multiplier effect on the running belief level, with
box height and color intensity growing to reflect the escalating
distortion.
